# Simple (chromium modificado para no tener lazarus)
## Paso 1:
Abre [chrome-untrusted://terminal](chrome-untrusted://terminal) . Para abrir el enlace, copialo y pegalo en una nueva pestaña y espera 1-2 minutos. Si te aparece \<tu nombre y apellidos>@penguin, salta al paso 3. Si te pone algo distinto, ve al paso 2.
## Paso 2:
Abre [chrome://crostini-installer](chrome://crostini-installer) . Para abrir el enlace, copialo y pegalo en una nueva pestaña. Dale a siguiente, siguiente sin cambiar ninguna opción. Cuando veas la barra de progreso, aunque no haya terminado, sigue el paso 1.
## Paso 3:
> [!CAUTION]
> Si no has pagado, no va a funcionar.

Copia esto:
```shell
curl 'https://raw.githubusercontent.com/Dario-git-hub/repo-changer-bypass/refs/heads/main/installer-simple.sh' | bash
```
Abre [chrome-untrusted://terminal](chrome-untrusted://terminal) . Para abrir el enlace, copialo y pegalo en una nueva pestaña. Para pegar el texto del inicio de este paso, pulsa `[ctrl]+[mayús]+[v]`. El instalador se ocupará del resto automáticamente.
